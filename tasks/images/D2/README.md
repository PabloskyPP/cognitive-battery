# D2 Task Images

This directory should contain the following images for the D2 attention and concentration test:

## Required Images:

- `ejemplo.png` - Example image showing the three target letters (d with two marks)
- `prueba.png` - Training/practice image with 22 numbered letters
- `fila1.png` through `fila14.png` - Main task images, each with 47 numbered letters

## Image Requirements:

Each image should contain letters with numbered hitboxes that can be clicked.
The letters should be d's and p's with various numbers of marks (lines) above/below them.

Target stimuli are: letter 'd' with exactly two marks in any position (both above, both below, or one above and one below).
